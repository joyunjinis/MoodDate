# MoodDate (In Progress)

연인 간 감정 상태 공유를 바탕으로 데이트 코스를 계획하고 기록할 수 있는 커플 전용 웹 서비스를 구현 중입니다.

## 🚧 Status
현재 본격 서비스를 이용하는 페이지인 메인 감정 상태 공유 페이지를 구현 중입니다.  

## ⚙️ Planned Features
- 로컬스토리지 및 서버 동기화 구조 설계
- 데이트 코스 계획을 위한 지도 API 연동
- 감정 상태 공유 기능 구현
- 커플 간 데이터 공유 상태 관리 로직 구현

## 🛠️ Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

> 이 프로젝트는 현재 개발 진행 중입니다.
